# Perbaikan Tampilan Menu Fitur v2.0.1

## Masalah
4 tombol teks sejajar horizontal (Riwayat, Template, Pelanggan, Laporan) terlalu sempit di layar HP — teks pecah jadi 2 baris, ikon+teks berdesakan, kelihatan berantakan.

## Solusi
Ganti dari 4 tombol horizontal menjadi **grid 2×2 kartu menu** (seperti dashboard app modern). Setiap kartu: ikon besar di atas, label pendek di bawah, dengan background biru muda dan sentuhan tactile.

### Perubahan layout `activity_main.xml`
Hapus block "MENU FITUR" LinearLayout horizontal yang lama, ganti dengan GridLayout 2×2 berisi 4 `MaterialCardView`:

```
┌─────────────┐  ┌─────────────┐
│   🕘        │  │   💾        │
│  Riwayat    │  │  Template   │
└─────────────┘  └─────────────┘
┌─────────────┐  ┌─────────────┐
│   👤        │  │   📊        │
│ Pelanggan   │  │  Laporan    │
└─────────────┘  └─────────────┘
```

- Pakai `androidx.gridlayout.widget.GridLayout` (sudah ada via material lib) dengan `columnCount="2"`
- Tiap kartu: ikon 28dp + label 13sp, padding 16dp, corner 12dp, `cardBackgroundColor=@color/blue_light`, tint ikon `@color/blue_primary`
- Label pakai teks pendek: "Riwayat", "Template", "Pelanggan", "Laporan" (sudah ada di strings.xml: `title_history`, `menu_templates`, `menu_customers`, `menu_report`)
- `android:foreground="?attr/selectableItemBackground"` untuk efek ripple saat ditekan
- Tinggi seragam lewat `layout_rowWeight=1` atau fixed height

### Perubahan kode
- `MainActivity.kt`: tidak perlu diubah (binding IDs `btnHistory`, `btnTemplates`, `btnCustomers`, `btnReport` tetap sama, hanya pindah dari MaterialButton ke kartu clickable — tetap pakai `setOnClickListener` di root card)
- Tidak ada perubahan Kotlin lain

### Cek samping
Sambil ada di layout, saya juga rapikan:
- Pastikan `btnSaveTemplate` (tombol "Simpan sebagai Template") tidak terlalu nempel ke tombol di atasnya — sudah ada `layout_marginTop="4dp"`, mungkin naikkan ke 8dp
